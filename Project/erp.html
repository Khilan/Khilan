the main aim of this system is that Admin can easily store information of the different
division like manufacturing management (MM), Material management (MM), Point Of Sale (POS).
And easily maintain this information edit, delete and update. This system is a powerful and
easy-to-use tool for the Member to store the information on internet and use from any place
through user name and password.
